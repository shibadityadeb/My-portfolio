Sentence Construction Tool

Completed: April 2025

A React + Vite-based web application where users complete sentences by selecting words from options, with a timer, scoring system, and feedback screen.

About the Project: Built using React+Vite, Tailwind CSS, JSON Server, and React Router, focusing on interactive sentence completion, real-time feedback, and smooth navigation.

Key Features:

  • Interactive fill-in-the-blank sentence game,with the ability to deselect answers.
  • "Next" button disabled until all blanks are filled.
  • Feedback screen showing correct/incorrect answers and final score.
  • Fully responsive design for a seamless experience on all devices.
  • Clean and minimal UI designed with Tailwind CSS.
