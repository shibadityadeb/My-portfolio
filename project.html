Tic-Tac-Toe Game

Completed: March 2025

An interactive Tic-Tac-Toe game built with React.js, featuring smooth UI and engaging animations.

About the Project: Developed using Vite + React.js, JavaScript, HTML, CSS, and canvas-confetti for celebratory effects.

Key Features:

  • Responsive grid-based UI for seamless gameplay.
  • Real-time game status updates (Win, Draw, Player Turn).
  • Confetti animation using canvas-confetti on player win.
  • Custom React hook for efficient state management.
  • Reset functionality for smooth gameplay experience.
